Mit dem Anti-Adblock Plugin von Adbreach können Sie Ihre Werbung Nutzern mit Adblocker auf Ihrer Wordpress Webseite anzeigen lassen. 


== Description ==

= Wichtig =

Das Plugin funktioniert nur mit einer [Registrierung auf Adbreach](https://adbreach.de/register), damit wir Ihnen Ihre Seite zuordnen können und diese daraufhin nach geblockten Bildern durchsuchen können.

= Allgemeine Beschreibung =
Haben sich auch Ihre Werbeeinnahmen durch die steigende Nutzung von AdBlockern vermindert? Dann sollten Sie in Zukunft Adbreach nutzen. Adbreach bietet Ihnen die Möglichkeit Werbeblocker zu blockieren und somit die Werbung auf Ihrer Webseite den Nutzern wieder anzuzeigen. 

Deshalb wird Werbung immer wichtiger. 
21 Prozent der Werbung wird in Deutschland durch Adblocker unterbunden. 
Mobile Werbeblocker auf Android und IOS werden bei den Nutzern immer beliebter. 
Die Werbepartner gehen weniger Partnerschaften mit Webseiten ein, welche Ihre Werbung blockieren lassen. 

= Globale Fakten und Trends =
Im Jahr 2015 ist die Nutzung von Werbeblockern um 41 Prozent gestiegen. In diesem Jahr ist durch die Adblocker ein wirtschaftlicher Schaden von 22 USD entstanden und der Trend ist weiterhin steigend. Aktuell nutzen bereits rund 180 Million Menschen Adblocker. 

= Nutzen =
Wir helfen Ihnen Ihr Werbepotential auszuschöpfen, indem keine Verluste mehr durch geblockte Werbeanzeigen entstehen. Das Plugin kann komplett kostenfrei heruntergeladen werden.

= Funktionen =
Nicht-blockbare Werbebilder erstellen: Adbreach speichert ihre Anzeigen dynamisch für Adblock Nutzer mit wechselnden Namen ab. So können Sie nicht vom Adblocker geblockt werden.

Popups trotz Adblock anzeigen lassen: Lassen Sie Ihre Popups nicht mehr geblockt. Mit der Registrierung können Sie Ihre Popup Werbeanzeigen verwalten.

= Support =
Wir kümmern uns so schnell wie möglich um Ihre Fragen und Anliegen. Sie können uns direkt über info@adbreach.de kontaktieren.

= Sicherheit =
Wir verändern weder Ihre Anzeigen noch Ihre Links oder sonstige Daten.


== Installation ==


1. Laden Sie das Plugin in das Verzeichnis `/wp-content/plugins/adbreach/`, oder installieren Sie es durch den Installieren-Button im Wordpress Plugin Reiter.
1. Aktivieren Sie das Plugin in der 'Plugins' Übersicht.
1. Fügen Sie Ihren persönlichen Verifizierungscode, den Sie durch eine [Registrierung auf Adbreach](https://adbreach.de/register) erhalten, ein.
1. Warten Sie maximal einen Tag bis Adbreach Ihre Webseite nach geblockten Bilder gescannt hat.

== Frequently Asked Questions ==

= Ist Adbreach kostenlos? =
Ja, die Standardversion ist kostenlos. Diese erlaubt bis zu 10 Werbeanzeigen auf einer Domain zu entblocken.

= Was ist der Unterschied zwischen der Standard- und der Businessversion? =
Im Gegensatz zur Standardversion, welche auf 10 Werbeanzeigen begrenz ist, bietet die Businessversion eine unbegrenzte Anzahl an Werbeanzeigen auf Ihrer Webseite. 

= Wie installiere ich Adbreach? =
Sie müssen sich lediglich kostenlos registrieren und entweder das WordPress Plugin herunterladen oder das Skript auf Ihrer Seite integrieren.  

= Welche Funktionen bietet Adbreach? =
Nicht-blockbare Werbebilder erstellen: Adbreach speichert ihre Anzeigen dynamisch für Adblock Nutzer mit wechselnden Namen ab. So können Sie nicht vom Adblocker geblockt werden.

Popups trotz Adblock anzeigen lassen: Lassen Sie Ihre Popups nicht mehr geblockt. Mit der Registrierung können Sie Ihre Popup Werbeanzeigen verwalten.

= Benötige ich spezielle Systemvorraussetzungen? =
Bei der Wordpress Version benötigen Sie lediglich eine aktuelle Wordpress Edition als Ihr Webseiten Back End.

= Wie lange dauert die Einrichtung von Adbreach? =
Sie benötigen lediglich ca. 1 Minute zur Einrichtung von Adbreach. 

= Werden meine Werbeanzeigen verändert? =
Wir verändern weder Ihre Anzeigen noch Ihre Links oder sonstige Daten.
